## 10. v3.32 — UHPS-full nasal preservation fix

**핵심 fix**: `_compose_with_jong` 옛한글 ㆁ → ㅇ remap.

**문제 분석**: ㆁ (옛이응 U+3181)은 표준 한글 완성형 받침 자리 (FINALS list)에 못 들어감. 따라서 V_NASAL 토큰이 받침 추가 시 None 반환 → 비음 정보 소실.

**해결**:
```python
def _compose_with_jong(syll, jong):
    ...
    # ㆁ → ㅇ for actual composition (표준 한글 받침 자리)
    if jong == 'ㆁ':
        jong = 'ㅇ'
    ...
```

**효과**:
- French nasal /ɛ̃/: `vɛ̃` → **ㅸ엥** (이전 ㅸ에)
- French nasal /œ̃/: `un` → **왱** (이전 왜)
- S31 KNOWN_GAP → 정상 case 승격

## 13. v3.35 — S40 Russian palatalized /tʲ/ /dʲ/ fix

**핵심 fix**: `_normalize_ipa` palatalization pre-process.

**문제**: 'ʲ' 마커가 REMOVE_DIACRITICS에서 strip되기 전에 'tʲ' 패턴 인식 필요.

**해결**:
```python
# NIKL palatalized stops: 'tʲ' → soft d (ㄷ), 'dʲ' → d (ㄷ)
s = s.replace('tʲ', 'd̥').replace('dʲ', 'd')
```

**효과**:
- S40 `ʐɨtʲ` → **ᄶ읃** (이전 ᄶ읕)
- **40/40 showcase 모두 정상 case** — known-gap 0 ✓
